# Detail Teknologi

Bagian ini membahas secara mendalam setiap teknologi RMI/RPC yang didemonstrasikan dalam proyek ini. Silakan pilih teknologi yang ingin Anda pelajari:

1.  [**gRPC**](./01-grpc.md): Standar industri modern untuk komunikasi antar layanan.
2.  [**XML-RPC**](./02-xmlrpc.md): Protokol klasik berbasis XML.
3.  [**RPyC**](./03-rpyc.md): Solusi transparan khusus Python.
4.  [**ZeroRPC**](./04-zerorpc.md): RPC cepat berbasis ZeroMQ.
5.  [**Pyro5**](./05-pyro5.md): Objek terdistribusi Python yang powerful.
6.  [**GraphQL**](./06-graphql.md): Query language fleksibel untuk API modern.
